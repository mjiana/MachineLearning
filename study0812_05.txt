----- 오차의 역전파(back propagation) : 오차를 역으로 전파한다.

hypo = X * W + b ===> X * W + b = hypo
    오차 : 라벨 - 예측값 : Y - hypo == Error의 크기
    최종오차  = 가중치 오차 + 절편오차 + 입력값 오차
    최종(전체) 오차는 각각의 오차의 합계와 같다. ===> 편미분 적용

    Error를 뒤로 보내서 W에서 발생하는 오차를 새로 구해서 할당
    b에서 발생하는 오차를 수정하여 새로운 b를 구하여 할당
다시 최종 오차를 구하고
이전 단계의 담당요소의 오차가 최소값이 될 때 까지 계속 반복하는 것.

순서 설명 :
    1. 임의의 초기 가중치를 준 뒤 결과를 계산한다.
    2. 계산결과와 원하는값 사이의 오차를 구한다.
    3. 경사하강법을 이용하여 바로 앞 가중치 오차가 작아지는 방향으로 업데이트 한다.
    4. 오차가 더이상 줄어들지 않을 때까지 1~3 과정 반복

공식 :
    새 가중치 = 현 가중치 - (오차변화/가중치변화)
        * (오차변화/가중치변화) = 가중치 기울기, d = 델타
    W(t+1) = Wt - (dE/dW)  ===> 가중치만 편미분한 것이다.
